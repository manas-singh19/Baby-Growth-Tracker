# Demo Video Script

## Duration: 2-3 minutes

### Introduction (15 seconds)
- Show app launch screen
- Display baby profile header: "Baby's Growth" with gender indicator
- Highlight WHO Standards badge

### Feature 1: Adding Measurements (45 seconds)

1. **Tap the + FAB button**
   - Show the measurement form appearing

2. **Fill in the form**
   - Date: Select today's date (or historical)
   - Weight: Enter 6.5 kg
   - Toggle to lb to show real-time conversion (14.33 lb)
   - Toggle back to kg
   - Height: Enter 65 cm
   - Head: Enter 41 cm

3. **Validation Demo**
   - Try entering invalid data (e.g., negative number)
   - Show error message appearing
   - Correct the data

4. **Save**
   - Tap Save button
   - Show success message
   - Return to charts screen

### Feature 2: Growth Charts (40 seconds)

1. **Weight Chart**
   - Show weight-for-age chart with percentile curves
   - Point out the 50th percentile line
   - Highlight baby's data point
   - Show the measurement details

2. **Switch Chart Types**
   - Tap "Height" button
   - Show height-for-age chart
   - Tap "Head" button
   - Show head circumference chart

3. **Chart Legend**
   - Scroll down to show legend
   - Explain percentile lines and baby's growth line

### Feature 3: History View (30 seconds)

1. **Navigate to History Tab**
   - Tap "History" tab
   - Show chronological list of measurements

2. **Measurement Details**
   - Point out date, age, values, and percentiles
   - Show color-coded percentiles (green for normal range)

3. **Edit Measurement**
   - Tap "Edit" on a measurement
   - Show form pre-filled with existing data
   - Make a small change
   - Save and show updated chart

4. **Delete Measurement**
   - Tap "Delete" on a measurement
   - Show confirmation dialog
   - Cancel to demonstrate safety

### Feature 4: Sample Data (20 seconds)

1. **Generate Sample Data**
   - Return to empty state (or show Demo button)
   - Tap "Demo" button
   - Confirm generation
   - Show 20 measurements appearing

2. **Chart with Multiple Points**
   - Show rich chart with growth trend
   - Scroll through history
   - Demonstrate smooth performance

### Feature 5: Data Persistence (15 seconds)

1. **Close and Reopen App**
   - Close the app completely
   - Reopen
   - Show all data still present
   - Highlight "no data loss"

### Conclusion (10 seconds)

- Quick recap of features:
  - ✅ Add/Edit/Delete measurements
  - ✅ WHO percentile charts
  - ✅ Unit conversion
  - ✅ Data persistence
  - ✅ Beautiful, responsive UI

- Show final screen with complete growth chart

## Recording Tips

1. **Use iOS Simulator or Android Emulator** for best quality
2. **Screen Recording**: Use QuickTime (Mac) or built-in screen recorder
3. **Resolution**: 1080p or higher
4. **Orientation**: Portrait mode
5. **Smooth Interactions**: Slow, deliberate taps and scrolls
6. **Highlight Key Features**: Pause briefly on important screens
7. **No Audio Needed**: Visual demonstration is sufficient
8. **Add Captions** (optional): Brief text overlays explaining each feature
